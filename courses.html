CST 8116
Introduction to Computer Programming (Java)

Level 01

Students develop introductory knowledge of computer programming with emphasis on problem analysis and design. They use the Java programming language to implement problem solving designs.

CST 8215
Introduction to Database

Level 01

Students learn the fundamentals of relational database design using Entity Relation Diagrams, and use SQL to create, modify and query a database. Students design and create databases that are maintainable, secure, and adaptable to change in business requirements.

CST 8101
Computer Essentials

Level 01

Study the essentials of computer systems and laptop management. Learn to configure a laptop environment and troubleshoot problems. Explore computer organization including basic numerical systems, and functional hardware and software components.

CST 8300
Achieving Success in Changing Environments

Level 01

Life in the 21st century presents opportunities, but it also creates ethical problems that need responsible solutions. Students apply critical thinking and decision-making tools to resolve some of these issues.

ENL 1813T
Communications I

Level 01

Communication remains an essential skill sought by employers, regardless of discipline or field of study. Using a practical approach, students develop stronger grammar and business writing skills to become effective communicators in various environments.

MAT 8001C
Tech Math for Computer Science

Level 01

Manipulate algebraic expressions, and solve exercises on algebraic equations and linear systems. Learn computer number systems, Boolean algebra and logic to solve problems involving computer systems. Add and subtract vectors using vector components.

CST 8284
Object Oriented Programming (Java)

Level 02

Learn object oriented programming methodology using the Java programming language. Object oriented concepts such as encapsulation, inheritance, abstraction and polymorphism are covered and reinforced with practical applications.

CST 8102
Operating Systems Fund (GNU/Linux)

Level 02

Learn the basic concepts and components of Operating Systems, and how they interact with hardware and software. Labs are designed to implement the theory by developing skills using the GNU/Linux command-line tools and utilities.

CST 2355
Database Systems

Level 02

Practical experience gained using market-leading object-relational database management systems like Oracle and MySQL. Students obtain hands-on experience with advanced engineering modeling tools, including programming with SQL and Oracle.

CST 8285
Web Programming

Level 02

Learn the basics of web programming, website design and implementation. Explore web-based solutions to problems of increasing interactivity and complexity. Practical assignments encourage students to build and maintain their own websites.

ENL 2019T
Technical Communications for Engineering Technology

Level 02

Students are exposed to exercises designed to foster independent and collaborative critical thinking, research, writing, visual communication and presentation skills related to technical topics.

GEP 1001
Cooperative Education Readiness

Level 02

Students participate in workshop-style sessions that prepare them to conduct a professional job search and succeed in the workplace. These sessions focus on cover letter and resume development, interview techniques and job search strategies.

CST 2335
Mobile Graphical Interface Programming

Level 03

Mobile devices play an instrumental part of everyday life, thus requiring knowledge of mobile graphical user interface development. Students learn how to program applications using the latest Android development tools. Topics include architecture, interface design, network communication, and database integration.

CST 8109
Network Programming

Level 03

Software programming demands an understanding of network topology and implementations. Students cover basic structure, design and layered communication models, with an emphasis on data communications, TCP/IP protocols, and socket programming.

CST 8390
Business Intelligence and Data Analytics

Level 03

Students examine the components of BI technology, and how it guides operational to strategic business decisions in the context of real-world applications. Data analytics techniques and statistical software are used to reveal patterns and trends concealed in raw data, yielding valuable insights.

CST 2234
Systems Analysis and Design

Level 03

Complex information technology systems require extensive planning and design. Guided by industry standard software engineering methodologies, students gain hands-on experience with case studies used to develop systems from inception through elaboration, construction and transition phases.

CST 8288
O.O.P with Design Patterns

Level 03

Design patterns are architectural solutions for common software challenges. Students apply object-oriented best practices using these patterns. They gain proficiency in SQL and JDBC for "data access objects." Through lab exercises, they enhance their skills in creating, debugging and deploying applications.
